Descrição

O curso de JavaScript visa ensinar os fundamentos da programação e as técnicas essenciais para criar páginas web dinâmicas e interativas, adicionando comportamento e lógica aos seus projetos.

Aqui você vai aprender a manipular elementos da página, responder a eventos do usuário, trabalhar com dados e criar funcionalidades complexas que rodam diretamente no navegador.

Para melhor compreensão, veja o vídeo de introdução.

Conteúdos

Introdução ao JavaScript e Variáveis. Estruturas Condicionais e de Repetição. Funções e Eventos. Manipulação do DOM (Document Object Model).

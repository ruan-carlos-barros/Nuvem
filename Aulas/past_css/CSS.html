Descrição

O curso de CSS visa ensinar os fundamentos e técnicas essenciais para a estilização de páginas web, transformando documentos HTML em interfaces visualmente atraentes e responsivas.

Aqui você vai aprender a abordar desde a sintaxe básica e seletores até a criação de layouts complexos e animações, dominando a linguagem que dá vida ao design da web.

Para melhor compreensão, veja o vídeo de introdução.

Conteúdos

Introdução ao CSS e Sintaxe Básica. Seletores e Propriedades Essenciais. Box Model: Margens, Bordas e Preenchimento. Layouts com Flexbox e Grid.
